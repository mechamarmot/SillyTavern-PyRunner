# PyRunner Development Notes

## SillyTavern Installation

App directory: `C:\ST-dev\SillyTavern`

## Paths

- UI Extension: `C:\ST-dev\SillyTavern\public\scripts\extensions\third-party\SillyTavern-PyRunner`
- Server Plugin: `C:\ST-dev\SillyTavern\plugins\pyrunner`

## Build & Deploy

```bash
npm run build
```

Then copy files to SillyTavern directories.
