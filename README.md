# SillyTavern-PyRunner

A SillyTavern extension for running Python scripts.

## Installation

Use the SillyTavern extension installer with this URL:
```
https://github.com/mechamarmot/SillyTavern-PyRunner
```

Or manually clone into your SillyTavern's `public/scripts/extensions/third-party` directory.

## Development

1. Clone the repository
2. Run `npm install`
3. Edit source code in `src` directory
4. Run `npm run build` to create the production bundle
5. The minimized bundle will appear in `dist`

> [!TIP]
> For live testing, clone the repo into `/public/scripts/extensions/third-party` of your SillyTavern installation.
